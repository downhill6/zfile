# P2P 文件传输

使用 React、WebRTC 和 PeerJS 构建的实时文件传输工具。创建或加入房间以立即与其他人分享你的文件。

## 功能

- 创建和加入文件传输房间。
- 实时共享文件。
- 简单易用的界面。


## 安装

1. 克隆仓库：

```bash
git clone https://github.com/downhill6/zfiles.git
```

2. 安装依赖：

```bash
cd zfiles
npm install
```

3. 启动开发服务器：

```bash
npm run dev
```
